# blackbean

The blackbean is a command tool for [elasticsearch](https://www.elastic.co/guide/en/elasticsearch/reference/7.x/index.html) operation by using cobra.

## shell completion
```console
# echo "source <(blackbean completion bash)" >> ~/.bashrc
```
